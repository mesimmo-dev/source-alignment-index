# Source Alignment Index

A lightweight framework for evaluating source alignment, contextual consistency, and reference coherence across structured analytical inputs.

## Purpose

The Source Alignment Index provides a compact review layer for comparing source materials against summaries, claims, generated outputs, or analyst notes.

The framework is intended as a minimal governance and evaluation artifact for AI-assisted review workflows, research validation, and human-in-the-loop assessment pipelines.

## Evaluation Dimensions

The framework uses three primary dimensions:

- **Source Alignment** — Degree of consistency between source material and derived output
- **Context Consistency** — Preservation of contextual meaning and framing
- **Reference Coherence** — Stability and relevance of supporting references

## Scoring Model

Each dimension is evaluated on a 0–5 scale.

| Score | Interpretation |
|---|---|
| 0–1.9 | Low alignment |
| 2–3.4 | Moderate alignment |
| 3.5–5 | Strong alignment |

## Example Use Cases

- AI-generated summary review
- Research memo validation
- Reference consistency checks
- Human review workflows
- Structured analytical assessment
